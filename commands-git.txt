Ligne de commande pour notre repository git-begins

➔ Cloner un repository

git clone "adresse du repository" 

➔ Les 3 commandes de push

    git add . 
    git commit -m "ton message"
    git push  

➔ La commande qui permet de pull

    git pull 

➔ Créer une branche (via terminal)

    git branch "nom de la branche"

➔ Changer de branche (via terminal)

    git checkout "nom de la branche"
